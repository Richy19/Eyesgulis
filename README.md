# a04 #

A04 encryption protocol as used by Exingulis

## Usage ##

```bash
# encode secret message
./encoder.py
# decode secret message
./decoder.py
```

### Flags ###

The following command line flags can be added to the commands:

* `--key='secret key'`
* `--msg='message to encode/decode'`
* `--hash='hashing algorithm'`
* `--verbose`
